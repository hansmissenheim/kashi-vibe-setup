# Installation

## Requirements

- Python 3.13 or higher

## Install from PyPI

```bash
pip install kashi
```

## Install from Source

```bash
# Clone the repository
git clone https://github.com/hansmissenheim/kashi.git
cd kashi

# Install with uv
uv sync

# Or install with pip
pip install -e .
```

## Verify Installation

```bash
kashi --version
```
